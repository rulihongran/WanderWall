# WanderWall

Web Project For 10002201 Web 2024 spring, Yunhai Team, Tongji University

## Set Up

### Frontend

```
npm install
npm run serve
```

### Backend

```
mvn spring-boot:run
```

### MinIO

1. Download MinIO

2. command:

   ```
   .\minio.exe server D:\Program Files\minio\data --console-address ":9001"
   ```
